# linq-citybike
uni project
## 5 queries
1. Top 10 station pairs by count.
2. Overnight rides by weekdays (when started).
3. Total time in given dataset spent riding at night grouped by member vs casual (night is 10PM-4AM).
4. Average duration by time block.
5. All destination from least popular station.
## Running
By defualt, to display the result of quries to stdout, we has to first to to the project directory and then run it
```
cd citybike
dotnet run
```
